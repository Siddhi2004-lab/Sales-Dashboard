# Sales-Dashboard
1) Project Title/ Headline:
   📈 Sales Performance Dashboard using Power BI
    A dynamic Interactive data visualization tool built to explore sales data worldwide.

2) Short Description/ Purpose:
   This dashboard presents an interactive Sales Performance Dashboard built using Power BI to analyze and visualize sales       data. The dashboard helps track key metrics such as total sales, profit, product performance, and regional sales trends,     enabling better business insights and data-driven decision making.

3) Tech Stack:
   The dashboard was built using the following tools and technologies:
   📊 Power BI – Data visualization and dashboard creation
   🔄 Power Query – Data cleaning and transformation
   🧮 DAX (Data Analysis Expressions) – Creating calculated measures and KPIs
   📁 Microsoft Excel / CSV – Data source and preprocessing

4) Data Source:
   The dataset used in this project contains sales transaction records, including information such as:
   📅 Order Date
   🛍️ Product Category
   💰 Sales Amount
   📈 Profit
   🌍 Region / Location
   👤 Customer Details

5) Features and Highlights:
   Key highlights of the dashboard include:
   📈 Sales Overview – Displays total sales, profit, and order metrics
   🌍 Regional Analysis – Visualizes sales performance across different regions
   🛒 Product Category Insights – Shows best-performing product categories
   📅 Time-based Trends – Tracks sales growth over months or years
   🎯 Interactive Filters – Allows users to filter data by category, region, or time period
   📊 Data Visualization – Uses charts, KPIs, and graphs for clear business insights
